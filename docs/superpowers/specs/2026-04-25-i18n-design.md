# 多语言支持（中英文切换）设计文档

## 目标

为项目添加中英文双语支持，默认根据操作系统语言自动识别，用户可在配置面板中手动切换并持久化偏好。

## 方案

采用 .NET 标准 .resx 资源文件方案，MAUI 原生支持。

## 资源文件结构

```
CodingPlanTimeRefresh/Resources/Strings/
├── AppResources.resx          # 默认资源（中文）
└── AppResources.en.resx       # 英文资源
```

- 默认 .resx 即中文，无需单独 `AppResources.zh.resx`
- 资源设计器由 Visual Studio/dotnet 自动生成 `AppResources.Designer.cs`

## AppConfig 变更

`AppConfig` 新增字段：

```csharp
public string? Language { get; set; }  // "zh" | "en" | null (自动)
```

`null` 表示自动检测，否则强制使用指定语言。

## LocalizationService

新建静态类 `LocalizationService`：

- **Initialize()**：读取 `AppConfig.Language`
  - 若为 null，根据 `CultureInfo.CurrentUICulture.TwoLetterISOLanguageName` 判断
  - 为 `"en"` 则设英文，否则默认中文
- **SetLanguage(string lang)**：切换 `CultureInfo.CurrentUICulture` 和 `CurrentCulture`，保存到 `AppConfig.Language`，触发 UI 刷新
- **CurrentLanguage** 属性：返回当前生效的语言代码

## XAML 绑定

使用 `x:Static` 绑定资源：

```xml
<Button Text="{x:Static resx:AppResources.SaveButton}" />
```

运行时切换语言后，`MainPage` 通过 `RefreshUI()` 方法重新为所有控件赋值。

## 配置面板变更

在配置浮层保存/取消按钮之前添加 Picker：

- 选项：自动 / 中文 / English
- 值：`auto` / `zh` / `en`
- 选择后立即生效并保存
