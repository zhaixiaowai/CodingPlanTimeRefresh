# 多语言支持（中英文切换）实现计划

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为项目添加中英文双语支持，默认根据操作系统语言自动识别，用户可在配置面板中切换并持久化偏好。

**Architecture:** 使用 .NET 标准 .resx 资源文件存储多语言字符串，新建 `LocalizationService` 管理语言检测与切换，修改 `AppConfig` 持久化语言偏好。运行时切换通过修改 `CultureInfo.CurrentUICulture` 实现并手动刷新 UI。

**Tech Stack:** .NET MAUI, .resx 资源文件, CultureInfo

```csharp
using System.Globalization;

namespace CodingPlanTimeRefresh;

public static class LocalizationService
{
    public static string CurrentLanguage { get; private set; } = "zh";

    public static void Initialize(string? savedLanguage)
    {
        string lang;
        if (!string.IsNullOrEmpty(savedLanguage) && savedLanguage != "auto")
        {
            lang = savedLanguage;
        }
        else
        {
            lang = CultureInfo.CurrentUICulture.TwoLetterISOLanguageName == "en" ? "en" : "zh";
        }

        ApplyCulture(lang);
    }

    public static string SetLanguage(string lang)
    {
        if (lang == "auto")
        {
            lang = CultureInfo.CurrentUICulture.TwoLetterISOLanguageName == "en" ? "en" : "zh";
        }
        ApplyCulture(lang);
        return lang;
    }

    private static void ApplyCulture(string lang)
    {
        CurrentLanguage = lang;
        var culture = new CultureInfo(lang == "en" ? "en" : "zh-CN");
        CultureInfo.CurrentUICulture = culture;
        CultureInfo.CurrentCulture = culture;
    }
}
```

private void OnLanguageChanged(object? sender, EventArgs e)

    var idx = LanguagePicker.SelectedIndex;
    var langCode = idx switch { 1 => "zh", 2 => "en", _ => "auto" };
